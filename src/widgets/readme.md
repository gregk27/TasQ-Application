# Widgets
This folder contains created Qt widgets and supporting code.

Each widget consists of 4 files, which should be placed in a corresponding subfolder.

Eg.
```
widgets/
    myWidget/
        myWidget.cpp
        myWidget.h
        myWidget.ui     
        ui_myWidget.h   <-- generated by uic, gitignored
    myOtherWidget/
        myOtherWidget.cpp
        myOtherWidget.h
        myOtherWidget.ui     
        ui_myOtherWidget.h
    ...
```
