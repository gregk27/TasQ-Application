Wooden Heads : Project Idea Submission

Created by Adi Nachmani on Oct 02, 2021

Our group aims to create a tool to help students stay organized. The software will mostly focus on the needs of university students, but its features will also be useful for high school students.

Users will be able to enter information into the system which will organize and display all data in an easy-to-track display. The software will have various features such as a schedule, reminder system, to-do list, sorting, and filters, profile feature, and a search function. Some of the functions listed are stretch goals and will be done only if the team deems it realistic. 

The schedule feature will provide users with both a list view or calendar view of their weekly class schedule, and will also allow them to add other events.
Users can add their upcoming assignments and tests which can be sorted or filtered according to the name, class, the due date, or weighting. All events entered in the calendar will have a "reminder" function where users can choose if and when they want to be reminded about that event. Users will be able to search by event or class name to display all relevant information or events. 

Users will be able to create To-Do Lists. These lists are separate from the events in the calendar and are intended for the users to jot down short-term tasks that are easily accessible. 

Each user will have a profile where they will be able to insert their name, age, school, and profile picture. This feature will be on the team's "stretch goal" list as it will require a more in-depth backend process. 

The software will integrate a tab design to allow users to easily switch between the class schedule, assignment pages, and to-do lists.
